🎬 CinoPhile - Movie Search Web App
CinoPhile is a sleek, responsive movie discovery platform built with Vanilla JavaScript, CSS3, and the OMDb API. It allows users to explore their cinematic tastes by searching for movie titles or filtering by popular genres.

✨ Key Features
Real-time Search: Instant movie fetching using the OMDb API.

Genre Filtering: Quick-access categories for Action, Romance, Thriller, and Documentary.

Responsive 3D Design: A modern, mobile-friendly grid layout featuring 3D hover effects and snappy UI interactions.

Dynamic UI: Automatically generates movie cards with titles, high-quality posters, and release years.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Flexbox & Grid).

Scripting: Vanilla JavaScript (ES6+), Async/Await, Fetch API.

Data Source: OMDb API.

Deployment: GitHub Pages.

🚀 How to Run Locally
Clone the repository:
git clone https://github.com/YOUR_USERNAME/cinophile-movie-app.git

Open index.html in your browser.

Enter a movie name in the search bar and hit Search!.
